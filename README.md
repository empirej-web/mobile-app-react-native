# Mobile App React Native
=========================

## Description
----------------

A cross-platform mobile app built using React Native, allowing users to create, edit and manage their tasks.

## Features
------------

* **Task Management**: Create, edit and delete tasks with ease
* **Task Priority**: Assign priority to tasks, with multiple levels of importance
* **Task Due Date**: Set due dates for tasks with reminders and notifications
* **Task List**: View and manage all tasks in one place
* **Task Filtering**: Filter tasks by priority, due date and status

## Technologies Used
-------------------

* **React Native**: Cross-platform framework for building native mobile apps
* **JavaScript**: Programming language for developing the app's business logic
* **Expo**: Library for building and deploying React Native apps
* **TypeScript**: TypeScript for better code organization and maintainability
* **Redux**: State management library for managing app state
* **React Navigation**: Library for managing app navigation

## Installation
------------

### Prerequisites

* Node.js 14.17.0 or higher
* npm 6.14.13 or higher
* Expo CLI 4.0.0 or higher

### Installation Steps

1. Clone the repository:
```bash
git clone https://github.com/username/mobile-app-react-native.git
```
2. Install dependencies:
```bash
npm install
```
3. Install Expo CLI:
```bash
npm install -g expo-cli
```
4. Start the app:
```bash
expo start
```
5. Run the app on a physical device or emulator using:
```bash
expo run android
or
expo run ios
```
